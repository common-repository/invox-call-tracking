== Description ==

The Invox Tracking Plugin allows you to easily add the Invox Tracking JavaScript tag to your WordPress website. This tag helps you track user interactions and gather analytics data.

For more information about INVOX, visit our [plugin homepage](https://invox.eu).

== Installation ==

1. Upload the `invox-call-tracking` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings in the WordPress admin panel.

== Configuration ==

To configure the plugin, go to 'Settings' > 'INVOX Call Tracking' in the WordPress admin panel. Enter your custom CID and Version (v) values and save the settings.

== Usage ==

Once configured, the INVOX Call Tracking JavaScript tag will be automatically added to your website.
